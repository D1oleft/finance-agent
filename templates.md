# 输出模板

## 记账确认

```
✅ 已记录
- {描述}：{金额}元（{账户}）
- 今日支出：{今日总额}元
- {账户}余额：{余额}元
- 本月{分类}：{本月分类总额}元
```

## 账户总览

```
💳 账户总览

支付宝：{余额}元
微信：{余额}元
银行卡：{余额}元
现金：{余额}元

总资产：{总额}元
本月支出：{本月支出}元
```

## 消费报表

```
📊 {月份}消费报表

总支出：{总额}元
总收入：{收入}元
结余：{结余}元

分类明细：
├─ 餐饮：{金额}元（{占比}%）{趋势}
├─ 交通：{金额}元（{占比}%）{趋势}
├─ 娱乐：{金额}元（{占比}%）{趋势}
└─ 购物：{金额}元（{占比}%）{趋势}

💡 分析：
- {分析1}
- {分析2}
```

## 预算概览

```
📊 {月份}预算

总预算：{预算}元
已支出：{已支出}元（{占比}%）
剩余：{剩余}元

分类预算：
├─ 餐饮：{已花}/{预算}元（{占比}%）{状态}
├─ 交通：{已花}/{预算}元（{占比}%）{状态}
└─ 购物：{已花}/{预算}元（{占比}%）{状态}

⚠️ 预警：
- {预警信息}
```

## 订阅列表

```
📺 订阅列表

月度订阅：
- {名称}：{金额}元/月（下次{日期}）

年度订阅：
- {名称}：{金额}元/年（下次{日期}）

月度成本：{总额}元
年度成本：{总额}元
```

## 定期收支

```
🔄 定期收支

支出：
- {描述}：{金额}元/月（每月{日}号，{账户}）

收入：
- {描述}：{金额}元/月（每月{日}号，{账户}）

每月净收入：{净收入}元
```

## 存钱目标

```
🎯 {目标名}

目标：{目标金额}元
已存：{已存金额}元（{占比}%）
剩余：{剩余金额}元
截止：{截止日期}

进度条：{进度条}

{预测信息}
```

## 负债总览

```
💳 负债总览

{负债名}：欠 {金额}元（还款日：{还款日}）

总负债：{总额}元
本月已还：{已还}元
本月待还：{待还}元
```

## 还款确认

```
✅ 还款成功

{负债名}：已还 {金额}元
剩余欠款：{剩余}元
下次还款日：{日期}
```

## 账单导入

```
📥 账单导入

平台：{平台}
时间范围：{开始日期} ~ {结束日期}
交易笔数：{笔数}笔
总支出：{支出}元
总收入：{收入}元

确认导入？（回复"确认"或"取消"）
```

## 导入完成

```
✅ 导入完成

已导入：{笔数}笔
跳过重复：{跳过数}笔
```

## 数据导出

```
📥 导出完成

时间范围：{开始日期} ~ {结束日期}
交易笔数：{笔数}笔
文件大小：{大小}

```
