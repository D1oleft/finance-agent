# 错误处理

## 首次使用

如果数据文件不存在：
1. 自动创建 `~/finance/data/` 目录
2. 从 `${CLAUDE_SKILL_DIR}/templates/` 复制模板文件
3. 提示用户设置账户和预算

```
👋 首次使用，已初始化数据目录

建议先设置：
1. 账户余额（/accounts 设置 支付宝 2500）
2. 月度预算（/budget 设置 3000）
```

## 金额无法识别

```
用户：吃饭了
Agent：请告诉我花了多少钱？例如：午饭25
```

## 多笔消费

```
用户：午饭25，咖啡18
Agent：请分开记录：
1. 午饭25
2. 咖啡18
```

## 账户不存在

```
用户：花呗买衣服200
Agent：花呗账户不存在。要添加吗？（回复"添加花呗"）
```

## 重复导入

导入账单时检测重复：
- 比对：时间 + 金额 + 描述
- 完全相同：跳过，提示"跳过重复 X 笔"
- 时间相同金额不同：提示确认

## 文件损坏

如果 JSON 文件格式错误：
```
⚠️ accounts.json 格式错误，已重置为默认值
```

## 预算超支

```
🚨 预算超支预警

购物分类已超预算！
预算：800元
已花：920元
超支：120元

建议：
1. 本月不再购物
2. 从其他分类借调预算
```

## 余额不足

```
⚠️ 余额不足

微信余额：50元
本次支出：100元

建议：
1. 切换其他账户
2. 先充值再消费
```
