# 数据导出

## 流程

### 1. 筛选数据
根据时间范围筛选交易记录。

### 2. 生成CSV
```csv
时间,类型,分类,描述,金额,账户
2026-06-16 12:30,支出,餐饮,午饭,25.00,微信
2026-06-16 18:00,支出,餐饮,晚饭,30.00,支付宝
```

### 3. 保存文件
保存到 `~/finance/reports/export/{日期}.csv`

### 4. 发送文件
```
📥 导出完成

时间范围：2026-06-01 ~ 2026-06-16
交易笔数：89笔
文件大小：12KB
文件路径：~/finance/reports/export/2026-06.csv
```

## 时间范围

- "本月" → 本月（默认）
- "上月" → 上月
- "2026-05" → 指定月份
- "全部" → 所有数据

## 边界情况

- 无交易记录 → 提示"暂无记录可导出"
- 时间范围过大 → 提示文件可能很大
