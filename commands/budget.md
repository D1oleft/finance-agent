# 预算管理

## 流程

### 1. 初始化检查
检查 `~/finance/data/budget.json` 是否存在，不存在则创建。

### 2. 操作类型

**查看（默认）**：
```
📊 6月预算

总预算：3,000元
已支出：2,100元（70%）
剩余：900元

分类预算：
├─ 餐饮：680/1,000元（68%）🟢
├─ 交通：180/300元（60%）🟢
├─ 娱乐：350/500元（70%）🟡
└─ 购物：800/800元（100%）🔴 超支！
```

**设置总预算**：
```
/budget 设置 3000
→ 月度预算已设置为 3,000元
```

**设置分类预算**：
```
/budget 餐饮 设置 1000
→ 餐饮预算已设置为 1,000元
```

### 3. 超支预警
当分类支出超过预算80%时，主动提醒：
```
⚠️ 预算预警

购物已花 720/800元（90%）
建议控制本月购物支出
```

## 边界情况

- 预算为0 → 提示设置预算
- 分类不存在 → 提示可用分类
- 超支 → 红色标记 + 预警
