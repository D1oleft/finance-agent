# 订阅管理

## 流程

### 1. 初始化检查
检查 `~/finance/data/subscriptions.json` 是否存在，不存在则创建。

### 2. 操作类型

**查看（默认）**：
```
📺 订阅列表

月度订阅：
- B站大会员：25元/月（下次6月18日）
- 网易云音乐：15元/月（下次6月20日）

年度订阅：
- 域名：100元/年（下次2027-01-15）

月度成本：46元
年度成本：652元
```

**添加**：
```
/sub 添加 B站大会员 25 月
→ 已添加订阅：B站大会员 25元/月
```

**删除**：
```
/sub 删除 B站大会员
→ 已删除订阅：B站大会员
```

**年度统计**：
```
📊 年度订阅成本

月度订阅：552元/年
年度订阅：100元/年
总计：652元/年

平均每月：54元
```

## 边界情况

- 订阅不存在 → 提示
- 重复添加 → 提示已存在
- 续费日期已过 → 提示续费
