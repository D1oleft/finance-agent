# 消费报表

## 流程

### 1. 读取数据
读取 `~/finance/data/transactions.jsonl`，按时间筛选。

### 2. 统计分析
- 按分类汇总
- 按账户汇总
- 计算趋势（与上月对比）

### 3. 输出报表

**月度报表**：
```
📊 2026年6月消费报表

总支出：3,200元
总收入：8,000元
结余：4,800元

分类明细：
├─ 餐饮：680元（21.3%）↑12%
├─ 交通：180元（5.6%）↓5%
├─ 娱乐：350元（10.9%）↑25%
└─ 购物：1,200元（37.5%）↑40%

账户支出：
├─ 支付宝：1,500元（46.9%）
├─ 微信：800元（25%）
└─ 银行卡：700元（21.9%）

💡 分析：
- 购物支出增长40%，主要来自618促销
- 餐饮支出正常
```

**日报**：
```
📝 今日消费（6月16日）

支出：58元
- 午饭：25元（微信）
- 咖啡：18元（支付宝）
- 地铁：5元（微信）

今日预算剩余：42元
```

## 时间范围

- "今天" → 今日
- "昨天" → 昨日
- "本周" → 本周
- "上周" → 上周
- "本月" → 本月（默认）
- "上月" → 上月
- "2026-05" → 指定月份

## 边界情况

- 无交易记录 → 提示"本月暂无记录"
- 时间范围过大 → 提示并建议缩小范围
