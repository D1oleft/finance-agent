# 账单导入

## 流程

### 1. 识别来源
- 用户发送文件 → 读取文件
- 用户说"导入支付宝" → 提示发送文件
- 用户说"导入微信" → 提示发送文件

### 2. 解析CSV

**支付宝格式**：
```
交易时间,交易分类,交易对方,商品说明,收/支,金额,收/付款方式,交易状态
```

**微信格式**：
```
交易时间,交易类型,交易对方,商品,收/支,金额(元),支付方式,当前状态
```

**智能识别**：如果格式不匹配，尝试按列解析。

### 3. 分类映射

| 支付宝/微信分类 | 本地分类 |
|----------------|---------|
| 餐饮美食 | 餐饮 |
| 交通出行 | 交通 |
| 日用百货 | 购物 |
| 生活服务 | 居住 |
| 娱乐 | 娱乐 |
| 医疗健康 | 医疗 |
| 教育培训 | 学习 |

### 4. 去重
比对：时间 + 金额 + 描述，相同则跳过。

### 5. 确认导入
```
📥 账单导入

平台：支付宝
时间范围：2026-05-01 ~ 2026-05-31
交易笔数：128笔
总支出：5,680元
总收入：8,200元

确认导入？（回复"确认"或"取消"）
```

### 6. 导入完成
```
✅ 导入完成

已导入：128笔
跳过重复：3笔
```

## 边界情况

- 文件格式错误 → 提示格式要求
- 无交易记录 → 提示文件为空
- 重复导入 → 自动去重
