# 提醒设置

## 流程

### 1. 操作类型

**设置提醒**：
```
/remind 设置 花呗还款 每月9号
→ 已设置提醒：花呗还款（每月9号）
```

**查看提醒**：
```
🔔 提醒列表

- 花呗还款：每月9号（支付宝）
- 信用卡还款：每月25号（银行卡）
```

**删除提醒**：
```
/remind 删除 花呗还款
→ 已删除提醒：花呗还款
```

### 2. 提醒触发
提醒依赖用户打开对话时检查。当用户打开对话时，系统检查是否有到期提醒，如有则主动推送：
```
⏰ 提醒

明天是花呗还款日
建议今天还款，避免逾期
```

> **注意**：提醒功能无法在后台自动运行。只有当用户主动打开对话时，才会检查并推送到期的提醒。

## 边界情况

- 提醒不存在 → 提示
- 重复设置 → 提示已存在
- 日期无效 → 提示正确格式
